Keysanity

For this year's freedom project, we use javascript and p5js to make Keysanity a typing game. The purpose of our game is to help users type faster and more efficiently. I have realized even I myself am not that fast at type. A typing game was a very good idea to help me get better at typing. Being able to type fast is very good. it can help you maintain time better. Not only does it help you type faster. Our game is super fun and interactive. We built this game mostly using javascript for functions and p5js for visuals.
