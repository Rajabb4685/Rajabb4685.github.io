copy cat project

If you look at my project as a student it’s where you view your grades. If you look at my project as a person who does not use the website then let me introduce you. This website is a website where students view their grades, transcript, past grade, etc. I made this project in December after completing applied visual design (AVD), CSS, and HTML. My goal was to make a copy of a website For example google homepage, apple home page, or delta. I choosed pupilpath because it’s one of the websites that I have viewed a lot. For this project i used many CSS and AVD in my style for example elements background color, margin, padding, width, display, divs, border radius, etc. I also used many HTML elements like paragraphs, heading underline elements, etc. Those elements and more helped me construct this project.
